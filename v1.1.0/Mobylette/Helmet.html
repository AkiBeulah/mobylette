Mobylette::Helmet

Mobylette::Helmet provides helper methods to help you when testing your controllers for a mobile request.

You must include Mobylette::Helmet into your test case, in the README there is more documentation about how to make this test wide.

Public Instance Methods

force_mobile_request_agent(user_agent = "Android") click to toggle source

Force the request for the user_agent

Remember to add it BEFORE the request

Example:

it "should render the mobile_device view on mobile request" do
  force_mobile_request_agent("Android")
  get :index
  response.should render_template(:mobile_device)
end

set_session_override(value) click to toggle source

set_session_override will set the 'value' to the session override control value may be:

  • :ignore_mobile -> This will disable mobile checking, and the original format will be rendered

  • :force_mobile -> This will force to all requests for this session be mobile (except xhr)

  • nil -> This will disable session override

# File lib/mobylette/helmet.rb, line 47
def set_session_override(value)
  session[:mobylette_override] = value
end
